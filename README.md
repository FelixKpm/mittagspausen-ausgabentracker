# 🍽 Mittagspausen-Ausgaben-Tracker

Eine einfache Web-App zum Tracken der täglichen Ausgaben während der Mittagspause.
Entwickelt im Rahmen der Ausbildung zum Fachinformatiker für Systemintegration bei EDEKA IT.

## Features

- **Einträge hinzufügen** mit Beschreibung, Betrag, Kategorie und Datum
- **Einträge bearbeiten und löschen**
- **Automatische Statistiken** für die aktuelle Woche, den aktuellen Monat und den Tagesdurchschnitt
- **Sortierung** nach Datum oder Betrag
- **Persistente Speicherung** via localStorage — Daten bleiben auch nach dem Schließen erhalten
- **Formularvalidierung** — der Button bleibt deaktiviert bis alle Felder ausgefüllt sind

## Technologien

- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (Vanilla)
- localStorage API
